# EduAccess Open Educational Resource Platform

## Project Description
EduAccess is an open-source web platform designed to aggregate and provide free access to educational resources for students in developing countries, particularly in Central Asia. The platform allows educators to publish learning materials and enables students to access them without financial or technological barriers.

## Technologies Used
* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **Deployment:** Docker support included

## Installation and Usage
To run this project locally:

1. Clone the repository:
   `git clone https://github.com/SerikErsultan/eduaccess.git`
2. Navigate to the project directory:
   `cd eduaccess`
3. Install dependencies:
   `pip install -r requirements.txt`
4. Run the application:
   `python app.py`
